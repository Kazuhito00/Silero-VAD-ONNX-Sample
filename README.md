# Silero-VAD-ONNX-Sample
[snakers4/silero-vad](https://github.com/snakers4/silero-vad)のONNX推論(PyTorch依存処理無し)サンプルです。

# Requirement
```
soundfile   0.13.1 or later
onnx        1.17.0 or later
onnxruntime 1.17.0 or later
```
